Тестовое задание

Многие задачи в нашей компании так или иначе связаны с анализом компаний РФ, поиском и сопоставлением различных атрибутов компаний в текстах. В этом задании мы предлагаем решить одну из таких задач: определить, что русское и английское названия принадлежат одной компании. 

Данные для задачи

Формат входных данных

В архиве лежат два файла: 

train_data.tsv - данные для обучения. Данные состоят из трех колонок:
1)	ru_name - российское название компании;
2)	eng_name - англоязычное название компании;
3)	answer - значение типа bool, которое показывает, относятся ли названия к одной компании.

test_data.tsv - данные в том же формате, без колонки “answer”, для которых значение этой колонки нужно предсказать.

Критерии оценки

1.	Для оценки финального решения будет использоваться F1 score.
2.	Чистота кода, оформление и понятность исследования


Требования к решению

В качестве решения мы ожидаем zip-архив со всеми *.py и *.ipynb файлами в папке solution и файлом result.tsv в корне. Файл result.tsv должен содержать единственную колонку типа bool с заголовком: answer.

В solution должно быть отражено исследование и весь код, необходимый для воспроизведения исследования.

Строгий формат имени zip-архива: LastName_FirstName.zip (пример Ivanov_Ivan.zip)
